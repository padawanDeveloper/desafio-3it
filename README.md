# Getting Started

> **Note**: Make sure you have completed the [React Native - Environment Setup](https://github.com/padawanDeveloper/afp-uno.git)

## Step 1: Instalar dependencias

```bash
npm install --legacy-peer-deps
cd ios
bundle install
pod install
```

## Step 2: Arrancar metro server

```bash
# using npm
npm start

# OR using Yarn
yarn start
```
